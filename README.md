# Hubble - Legal Timeline Visualizer

AI-powered tool for extracting timeline events from legal documents and generating Gantt chart visualizations.

## Architecture

- **Frontend**: React + Vite + Tailwind CSS + shadcn/ui
- **Backend**: Python FastAPI + Claude 3.5 Sonnet + matplotlib
- **Features**:
  - PDF document upload
  - Real-time AI extraction with streaming progress
  - Gantt chart visualization (NexVira template style)
  - PNG export

## Setup

### Backend

1. Navigate to backend directory:
   ```bash
   cd backend
   ```

2. Install Python dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Configure API key:
   - Edit `backend/.env`
   - Add your Anthropic API key:
     ```
     ANTHROPIC_API_KEY=sk-ant-your-actual-key-here
     ```

4. Start backend server:
   ```bash
   python main.py
   ```
   Server will run on [http://localhost:8000](http://localhost:8000)

### Frontend

1. Navigate to frontend directory:
   ```bash
   cd hubble
   ```

2. Install dependencies (if not already done):
   ```bash
   npm install
   ```

3. Start dev server:
   ```bash
   npm run dev
   ```
   Frontend will run on [http://localhost:5173](http://localhost:5173)

## Usage

1. Open [http://localhost:5173](http://localhost:5173)
2. (Optional) Type a request like "analyze executive timeline" or "show regulatory events"
3. Click the upload icon and select a PDF document
4. Watch the AI thinking process in real-time on the left panel
5. View the generated timeline chart on the right panel
6. Click "Export PNG" to download the visualization

## Event Extraction Schema

The AI extracts structured events with:
- **actor**: Who performed the action
- **action**: What they did
- **target**: Who/what was affected
- **roleType**: Their role or department (AI-generated)
- **start_date**: When it started
- **end_date**: When it ended (if applicable)
- **context**: Why or under what circumstance
- **milestone**: Whether it's a major event

## Example Use Cases

- "Create a timeline of all executives at the company"
- "Show when regulatory investigations occurred"
- "Visualize the lifecycle of key contracts"
- "Track stakeholder involvement over time"

## Color Palette

Colors are dynamically generated by the AI based on extracted role types. No hardcoded categories.
